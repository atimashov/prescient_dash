DATA API

A set of APIs for use by dashboard for displaying data.

API

Sales

This page return information about following info MONTHLY, WEEKLY, DAILY, HOURLY and by DAY OF THE WEEK separated by weekend or weekday
  • date
  • sales (registrations)
  • dockets
  • average purchase
    • link: .../sales

    Subscribers

    This page return information about following info MONTHLY
  • date
  • new customers (registrations)
  • new buyers
  • buyers in DB cumulative
    • link: .../subscribers

    Products

    There are 2 links are represented (monthly and daily separately). These pages return following info:
  • date
  • product (only for monthly page for top 5 (!!!) products)
  • sales
  • dockets
    • links: .../products & .../products_monthly

    Stores

    This page returns information about stores with the most sales such as
  • store ID
  • store name
  • sales
  • dockets
    • link: .../stores

    FILTERS

     Possible parameters:
  • from (date)
  • to (date)
  • region
  • states
  • product
  • plot_type (monthly, daily, weekdaily, hourly: applicable for some types of API)
